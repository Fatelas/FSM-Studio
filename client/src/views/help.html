Parsing Language

Data Formats

  • Decimal 
    
    count = 10;
    // Assigns the decimal value 10 to the 'count' variable.
  • Binary 
    
    count = 0b1010;
    // Assigns the binary value 1010 (decimal 10) to 'count'.
  • Hexadecimal 
    
    count = 0xA;
    // Assigns the hexadecimal value A (decimal 10) to 'count'.
  • Boolean 
    
    flag = true;
    // Assigns the boolean value 'true' to 'flag'.

Operators

Arithmetic Operators

  • Addition ('+') 
    
    i = 2+2;
    // Assigns the value 4 to 'i' which is the result of the operation 2+2
  • Subtraction ('-') 
    
    i = 3-2;
    // Assigns the value 1 to 'i' which is the result of the operation 3-2

Comparison Operators

  • Equal to ('==') 
    
    i == 1;
    // Compares the value of 'i' with the value '1'
  • Not equal to ('!=') 
    
    i != 1;
    // Compares the value of 'i' with the value '1'
  • Greater than ('>') 
    
    i > 1;
    // Checks if the value of 'i' is greater than '1'
  • Less than ('<') 
    
    i < 1;
    // Checks if the value of 'i' is less than '1'
  • Greater than or equal to ('>=') 
    
    i >= 1;
    // Checks if the value of 'i' is greater than or equal to '1'
  • Less than or equal to ('<=') 
    
    i <= 1;
    // Checks if the value of 'i' is less than or equal to '1'

Logical Operators

  • AND ('&&') 
    
    i == 1 && j == 2;
    // Checks if the expressions "i==1" and "j==2" both return true
  • OR ('||') 
    
    i == 1 || j == 2;
    // Checks one of the expressions "i==1" and "j==2" returns true
  • NOT ('!') 
    
    !(i);
    // Negates the value of i

Assignment Operators

  • Assignment ('=') 
    
    i = 3;
    // Assigns the value 3 to 'i'

Bitwise Operators

  • Bitwise AND ('&') 
    
    a = 10;
    b = 5;
    c = a & b;
    // Performs a bitwise AND operation on each pair of corresponding bits of the operands.
  • Bitwise OR ('|') 
    
    a = 10;
    b = 5;
    c = a | b;
    // Performs a bitwise OR operation on each pair of corresponding bits of the operands.
  • Bitwise XOR ('^') 
    
    a = 10;
    b = 5;
    c = a ^ b;
    // Performs a bitwise XOR (exclusive OR) operation on each pair of corresponding bits of the operands.
  • Bitwise NOT ('~') 
    
    a = 10;
    c = ~a;
    // Inverts the bits of the operand, changing 0s to 1s and 1s to 0s.
  • Left shift ('<<') 
    
    a = 10;
    b = 5;
    c = a << b;
    // Shifts the bits of the left operand to the right by the number of positions specified in the right operand, filling with the sign bit.
  • Right shift ('>>') 
    
    a = 10;
    b = 5;
    c = a >> b;
    // Shifts the bits of the left operand to the right by the number of positions specified in the right operand, filling with the sign bit.

Keywords

  • Else
    The else keyword is used in conditional statements. It defines the transition to be executed if the other conditions specified for a certain state are invalid. In other words, it provides an alternative course of action when the condition is not met.
  • True
    True is a boolean value that represents a logical "true" or "yes" condition. It is one of two possible boolean values, with the other being false. "true" is often used in conditional expressions, comparisons, and logical operations to represent that a condition is met or that a statement is valid or correct.
  • False
    False is the opposite of true and represents a logical "false" or "no" condition. It is the other boolean value and is used to indicate that a condition is not met or that a statement is invalid or incorrect.
